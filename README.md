# [PDB Plot](https://pdb.surge.sh)
An interactive period-doubling bifurcation plotter! Currently plots logistic map.

## Tools
- Babel
- SCSS
- Typescript
- Vue.js
- Webpack

## Setup
Start live development server:
```sh
npm install
npm run start:hmr
```

## Build
Build for deployment (output in `dist/`):
```sh
npm run build:prod
```
